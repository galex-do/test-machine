  • Disabled Test Case Status

Default Status: Active

Status Concept Explanation
Test Case Status

Represents whether a test case is available for use:

  • Active: Test case is ready and available for execution
  • Disabled: Test case is not available for execution (archived/deprecated)
Test Run Status

Represents the result of executing a test case:

  • Pass: Test execution succeeded
  • Fail: Test execution failed
  • Blocked: Cannot execute due to dependency
  • Skip: Test was skipped
  • Not Executed: Test hasn't been run yet
